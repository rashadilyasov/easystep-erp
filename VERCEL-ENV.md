# Vercel Environment Variables

Vercel-da **Settings** → **Environment Variables** əlavə edin:

| Name | Value |
|------|-------|
| `NEXT_PUBLIC_API_URL` | `https://api.easysteperp.com` |

> **Vacib:** Bu olmadan qeydiyyat və login işləməz. Dəyişəndən sonra **Redeploy** edin.

## API xətası alırsınızsa?

1. Vercel-də `NEXT_PUBLIC_API_URL` = `https://api.easysteperp.com` olmalıdır
2. Vercel **Redeploy** edin
3. Railway API Online olmalıdır (`api.easysteperp.com`)
